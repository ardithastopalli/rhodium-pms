Description

RhodiumPMS is a payroll management system developed in meteorJS and bootstrap.It sums up all financial records of salaries for an employee, wages, bonuses and deductions.

Features

    Record attendance patterns
    Graphs
    Encrypted passwords(blowfish)
    Fast search (elastic search)
